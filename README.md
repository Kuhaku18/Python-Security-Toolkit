# 🛡️ Python Security & OSINT Toolkit

Welcome to my cybersecurity portfolio. This repository contains 4 custom-built command-line interface (CLI) tools designed for network reconnaissance, offensive security testing, and cryptographic analysis.

These scripts demonstrate proficiency in Python network sockets, external API integration, HTTP request handling, and hashing algorithms.

## 📁 The Arsenal

* **`hash_cracker.py`**: A brute-force password cracker utilizing the `hashlib` library to reverse MD5 hashes via dictionary attacks.
* **`OSINT IP Tracker.py`**: A reconnaissance tool that queries external REST APIs to geolocate target IP addresses and map network infrastructure.
* **`The Directory Buster.py`**: A web enumeration script that automates HTTP requests to discover hidden directories and bypass soft-404 responses.
* **`The Port Scanner.py`**: A custom network scanner built with the `socket` library to identify open ports on remote servers.
